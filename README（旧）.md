# CSV診断ツール

## CSV診断ツールについて

このツールは、CSVファイルを読み込み、集計や分析を行う前にデータの状態を確認するための診断ツールです。

主に以下の内容を確認できます。

- 読み込んだCSVファイル
- 行数・列数
- 列名一覧
- データ型
- 欠損値の数
- 重複行数
- 数値列候補
- 日付列候補
- 診断コメント

## 現在の主な機能

### 1. CSVファイルの読み込み

`input` フォルダ内のCSVファイルを読み込みます。

現在は、`main.py` 上部の `INPUT_FILE_NAME` を変更することで、読み込むCSVファイルを切り替えられます。

例：

```python
INPUT_FILE_NAME = "sample.csv"
```

または、

```python
INPUT_FILE_NAME = "sample_clean.csv"
```

### 2. 基本情報の確認

CSVの行数、列数、列名、データ型を確認できます。

### 3. 欠損値の確認

各列に欠損値がいくつあるかを確認できます。

例：

```text
数量: 1
単価: 2
```

### 4. 重複行の確認

CSV内に重複行があるか確認できます。

例：

```text
重複行数: 1
```

### 5. 数値列候補の判定

数値として扱えそうな列を自動で判定します。

例：

```text
数量
単価
```

### 6. 日付列候補の判定

列名に「日付」または「date」が含まれる列を対象に、日付として扱えそうか確認します。

これにより、日付ではない文字列列に対して無理に日付変換を行わないようにしています。

### 7. 診断コメントの出力

レポートの最後に、CSVの状態をまとめた診断コメントを出力します。

例：

```text
## 9. 診断コメント
- 総合判定: 要確認
- 欠損値が含まれる列があります: 数量, 単価
- 重複行が1行あります。
- 日付列候補があります: 日付
- 数値列候補があります: 数量, 単価
```

欠損値や重複行がないCSVの場合は、次のように表示されます。

```text
- 総合判定: 大きな問題は見つかりませんでした
```

## フォルダ構成

```text
csv_diagnosis_tool/
├─ main.py
├─ input/
│  ├─ sample.csv
│  └─ sample_clean.csv
├─ output/
│  └─ diagnosis_report.md
└─ README.md
```

## 実行方法

VS Codeで `csv_diagnosis_tool` フォルダを開き、ターミナルで次を実行します。

```powershell
py main.py
```

実行すると、診断結果が以下のファイルに出力されます。

```text
output/diagnosis_report.md
```

## 現在の学習ポイント

このツール作成を通して、以下を学習しています。

- `Path` を使ったファイルパスの管理
- `pandas` を使ったCSV読み込み
- 欠損値の確認
- 重複行の確認
- 数値列・日付列の判定
- `if` 文による条件分岐
- `for` 文による列ごとの繰り返し処理
- `continue` による処理のスキップ
- Markdown形式のレポート作成
